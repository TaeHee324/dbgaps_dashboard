| `test_metrics.py` | `src/metrics.py` 순수 함수 단위 테스트. 현재도 유효. | **유지 필수** |
| `test_smoke_engine.py` | 샘플 데이터로 전체 엔진 smoke 테스트. | **유지 필수** |
| `conftest.py` | sample_prices, sample_etf_master, sample_trades, base_weights_path 픽스처. | **유지 필수** |
| `test_output_schema.py` | output/ CSV 컬럼 계약 검증. **레거시 잔재 1개 포함**: `DATA_LOADER = ROOT / "web" / "data_loader.py"` 참조가 있으나, 해당 테스트(`test_data_loader_files_are_schema_covered`)는 이미 `@pytest.mark.skip(reason="web/data_loader.py removed in FastAPI migration")`으로 스킵 처리됨. 나머지 스키마 검증 테스트는 현재도 유효. | **유지 필수** (레거시 참조는 스킵 처리로 무해함) |

`tests/` 폴더 전체 삭제 불가. `test_boundaries.py`는 CI/CD 안전망으로 특히 중요.

**소소한 기술 부채**: `test_output_schema.py` 내 `web/data_loader.py` 참조 변수(`DATA_LOADER`)와 스킵된 테스트 함수는 언젠가 정리하면 좋으나 급하지 않음.

### 5-2. `.env.example` — **삭제 불가, 유지 필수**

```
DATABASE_URL=postgresql://postgres:<password>@<host>:<port>/railway
```

CLAUDE.md 환경 설정 섹션에서 첫 번째 온보딩 단계로 명시:
```bash
cp .env.example .env
# Edit .env: DATABASE_URL=postgresql://...
```

`DATABASE_URL` 없으면 `run_engine.py`와 ETF 포트폴리오 페이지가 실패함. 신규 개발자 진입점이므로 삭제 불가.

### 3-3. `GET /api/risk/etf-analysis`

```
반환: List[{
  code: str,
  name: str,
  current_weight: float,            # ETF시가 합 기준 현재 비중 (소수)
  target_weight: float | None,      # trade_log 최신 weight_after (소수)
  weight_drift: float | None,       # current_weight - target_weight
  individual_mdd: float,            # 전체 가격 이력 기준 개별 MDD (음수, 소수)
  current_drawdown: float,          # 현재 고점대비 낙폭 (음수, 소수)
  vol_20d: float | None,            # 최근 20거래일 변동성 연율 (소수)
  risk_contribution_pct: float | None,  # 위험기여도 0~1 (소수). 데이터 부족 시 None
}]
정렬: risk_contribution_pct 내림차순 (None은 하단)
```

계산 상세:
```python
holdings = _get_fifo_holdings()
target_weights = _get_target_weights()   # {code: weight_after(소수)}

# prices_daily.csv에서 보유 코드만 필터
held_codes = [h["code"] for h in holdings if h["quantity"] > 0]
prices_df = pd.read_csv(...)
prices_df = prices_df[prices_df["code"].isin(held_codes)]
price_pivot = prices_df.pivot_table(index="date", columns="code", values="close").ffill()

total_mv = sum(h["market_value"] for h in holdings)
result = []
for h in holdings:
    if h["quantity"] <= 0:
        continue
    code = h["code"]
    current_weight = h["market_value"] / total_mv if total_mv > 0 else 0.0
    target_weight = target_weights.get(code)  # None 가능
    weight_drift = (current_weight - target_weight) if target_weight is not None else None

    # 개별 MDD / 현재낙폭
    if code in price_pivot.columns:
        closes = price_pivot[code].dropna()
        rolling_max = closes.cummax()
        drawdowns = (closes - rolling_max) / rolling_max
        individual_mdd = float(drawdowns.min())
        current_drawdown = float(drawdowns.iloc[-1])
    else:
        individual_mdd = 0.0
        current_drawdown = 0.0

    # 20일 변동성
    vol_20d = None
    if code in price_pivot.columns:
        rets = price_pivot[code].dropna().pct_change().dropna()
        if len(rets) >= 10:
            vol_20d = float(rets.tail(20).std() * np.sqrt(252))

    result.append({..., "code": code, "current_weight": current_weight,
                   "target_weight": target_weight, "weight_drift": weight_drift,
                   "individual_mdd": individual_mdd, "current_drawdown": current_drawdown,
                   "vol_20d": vol_20d, "risk_contribution_pct": None})  # 기여도는 아래에서 채움

# Risk Contribution (보유 ETF 전체 동시 계산)
if len(held_codes) >= 1:
    rets_1y = price_pivot[held_codes].pct_change().dropna().tail(252)
    if len(rets_1y) >= 20:
        w_arr = np.array([h["market_value"] for h in result]) / total_mv
        cov = rets_1y.cov().values * 252
        port_var = float(w_arr @ cov @ w_arr)
        if port_var > 0:
            pct = (w_arr * (cov @ w_arr)) / port_var  # 합 = 1.0
            for i, item in enumerate(result):
                item["risk_contribution_pct"] = float(pct[i])
```

### 3-4. `GET /api/risk/stress-test`

```
반환: List[{
  scenario: str,
  shock_desc: str,
  estimated_loss_pct: float,        # 손실률 = 손실액 / (ETF+현금) (소수, 음수)
  estimated_loss_krw: float,        # 손실 금액 (원, 음수)
  individual_limit_breach: bool,    # 충격 후 ETF 20% 상한 위반

```

시나리오 4종:
| # | scenario | shock_desc | 적용 조건 |
|---|----------|-----------|----------|
| 1 | `전체 시장 하락` | `전 ETF -10%` | 전체 |
| 2 | `위험자산 집중 충격` | `위험자산 ETF -15%` | `risk_type == "위험"` |
| 3 | `초기 운용 급락 (SORR)` | `전 ETF -20%` | 전체 |
| 4 | `금리 상승 충격` | `채권형 ETF -8%` | `"채권" in asset_class` |

현금 계산:
```python
# _get_fifo_holdings()에서 누적 추적된 buy_cost, sell_proceeds 활용
cash = INITIAL_CAPITAL - total_buy_cost + total_sell_proceeds
total_portfolio = total_mv + cash  # 전체 포트폴리오

# 충격 계산 예시
shocked_mv = {}
for h in holdings:
    rate = shock_rate if 조건 else 0.0
    shocked_mv[h["code"]] = h["market_value"] * (1 + rate)

loss_krw = sum(shocked_mv.values()) - total_mv  # 음수
loss_pct = loss_krw / total_portfolio            # 음수

# 비중 재계산 → 규칙 위반 체크
new_total_mv = sum(shocked_mv.values())
if new_total_mv > 0:
    new_weights = {c: v / new_total_mv for c, v in shocked_mv.items()}
    individual_breach = any(w > INDIVIDUAL_LIMIT for w in new_weights.values())
    risk_weight = sum(w for c, w in new_weights.items()
                      if risk_map.get(c, "") == "위험")
    risk_breach = risk_weight > RISK_ASSET_LIMIT
```

- 배경: `#FFFBEB` (amber-50), 테두리: `#FCD34D`

### 6-3. 섹션 2 — ETF별 리스크 테이블

컬럼 구성:
| 컬럼 | 너비 | 정렬 | 비고 |
|------|------|------|------|
| ETF명 | 고정 | 좌 | name + (code) 서브텍스트 |
| 현재비중 | auto | 우 | % (소수→퍼센트) |
| 목표비중 | auto | 우 | % (null → —) |
| 이탈폭 | auto | 우 | +6.2%p / -5.8%p (color) |
| 개별 MDD | auto | 우 | % (빨강) |
| 현재낙폭 | auto | 우 | % (빨강, 손절선 참고) |
| 20일 변동성 | auto | 우 | % (null → —) |
| 위험기여도 | auto | 우 | % (null → —) |

**셀 색상 규칙:**
```
개별 MDD 셀:
  항상 #DC2626 텍스트 (손실 의미)

현재낙폭 셀:
  낙폭 > -10%: 정상 (기본 텍스트)
  -10% ~ -15%: amber 배경 (#FEF3C7) + #D97706 텍스트
  < -15%: red 배경 (#FEE2E2) + #DC2626 텍스트

이탈폭 셀:
  +5%p 이상 (과중): #DC2626 + "▲" prefix
  -5%p 이하 (과소): #D97706 + "▼" prefix
  범위 내: #64748B

위험기여도 셀:
  risk_contribution_pct > current_weight * 2:
    amber 배경 (#FEF3C7) + "!" suffix + title="비중 대비 위험 집중"

행 전체:
  현재낙폭 < -15% → 행 전체 배경 #FFF1F2 (연한 붉은색)
```

정렬: 기본 `risk_contribution_pct` 내림차순, null은 하단

### 6-4. 섹션 3 — 스트레스 테스트 (2×2 그리드)

```
┌──────────────────────┐  ┌──────────────────────┐
│ 전체 시장 하락        │  │ 위험자산 집중 충격    │
│ 전 ETF -10%          │  │ 위험자산 ETF -15%     │
│                      │  │                      │
│ 손실 -97,234만원     │  │ 손실 -63,120만원     │
│ 손실률    -9.7%      │  │ 손실률    -6.3%      │
│                      │  │                      │
│ ETF 20% 위반  ✓없음  │  │ ETF 20% 위반  ✓없음  │
│ 위험자산 위반  ✓없음  │  │ 위험자산 위반  ✗있음  │
```

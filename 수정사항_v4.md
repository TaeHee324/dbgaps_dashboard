## 배경 및 현재 상태

운용현황 페이지(`/`) 하단의 **일별 수익률 히트맵** 카드에 두 가지 문제가 있다.

### 문제 1: 좌측 월별 수익률 바 차트가 삭제되지 않음

`수정사항_v2.md`에서 "카드 왼쪽 월별 수익률 바 차트 삭제, 남은 공간을 히트맵으로 꽉 채우기" 를 요청했으나 미반영 상태.

현재 레이아웃 구조 (`DailyHeatmap.tsx` 318~381 라인):
```tsx
<div style={{ display: "grid", gridTemplateColumns: "220px 1fr", ... }}>
  {/* 좌측: MonthlyBarChart — 삭제 대상 */}
  <div style={{ paddingRight: 12, borderRight: "1px solid #E4E9EF" }}>
    <MonthlyBarChart data={barChartData} />
  </div>
  {/* 우측: 월 탭 + 달력 히트맵 */}
  <div style={{ paddingLeft: 12 }}>
    ...
  </div>
</div>
```

### 문제 2: 기본 선택 월이 오늘 날짜 기준이 아님

현재 코드 (272번 라인):
```ts
const activeMonth = selectedMonth ?? months[months.length - 1] ?? null;
```

`months[months.length - 1]`은 데이터에서 추출한 배열의 마지막 요소를 그대로 쓴다.
탭을 클릭해 과거 월로 이동한 후 새로고침하면 의도치 않게 다른 월이 기본값이 될 수 있다.
오늘 날짜(2026-05-27) 기준으로 **가장 최근 이하 월**을 명시적으로 선택해야 한다.

## 수정 계획 (4개 변경)

### 변경 1: MonthlyBarChart import 삭제
```tsx
// 삭제
import { MonthlyBarChart } from "@/components/charts/MonthlyBarChart";
```

### 변경 2: monthlyQuery + barChartData 삭제

삭제 대상 코드 블록 (`DailyHeatmap.tsx` 257번, 293~298번 라인):
```tsx
// 삭제
const monthlyQuery = useMonthlyReturns();

// 삭제
const barChartData = useMemo(() => {
  return (monthlyQuery.data ?? []).map((mr) => ({
    time: `${mr.year}-${String(mr.month).padStart(2, "0")}-01`,
    value: mr.monthly_return,
  }));
}, [monthlyQuery.data]);
```

`useMonthlyReturns` import도 같이 정리:
```tsx
// 변경 전
import { useBacktestNav, useMonthlyReturns, type NavPoint } from "@/lib/hooks/dashboard";
// 변경 후
import { useBacktestNav, type NavPoint } from "@/lib/hooks/dashboard";
```

### 변경 3: 레이아웃 — 2단 그리드 → 히트맵 단일 영역

```tsx
// 변경 전 (322~379번 라인)

    <MonthlyBarChart data={barChartData} />
  </div>
  {/* 우측: 월 탭 + 달력 히트맵 */}

    ...월 탭 + CalendarGrid...
  </div>

// 변경 후
<div style={{ padding: "12px 14px" }}>
  ...월 탭 + CalendarGrid (그대로 유지)...

```

### 변경 4: 기본 월 선택 — 오늘 날짜 기준으로

```tsx
// 변경 전
const activeMonth = selectedMonth ?? months[months.length - 1] ?? null;

// 변경 후
const today = new Date();
const thisMonth = `${today.getFullYear()}-${String(today.getMonth() + 1).padStart(2, "0")}`;
const defaultMonth =
  [...months].reverse().find((m) => m <= thisMonth) ?? months[months.length - 1];
const activeMonth = selectedMonth ?? defaultMonth ?? null;
```

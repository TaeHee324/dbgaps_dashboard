# UI_GUIDE.md — 폐기됨

> **이 파일은 Streamlit 구현 기준으로 작성된 구식 가이드입니다.**
> 현재 스택(Next.js 15 + Tailwind CSS + TanStack Query)과 무관한 내용을 담고 있습니다.
> 참고 가치가 없으므로 삭제를 권장합니다.

## 현재 UI 관련 문서

프론트엔드 UI 작업 전 아래 문서를 참고하세요:

| 문서 | 역할 |
|------|------|
| `docs/DESIGN.md` | 제품 성격 및 시각 방향 (색상, 타이포, 레이아웃, 금지 패턴) |
| `docs/DESIGN-LANGUAGE.md` | 디자인 판단 규칙 및 안티패턴 체크 |
| `docs/design-tokens.json` | 색상/타이포 등 머신 가독 토큰 |
| `docs/QA_CHECKLIST.md` | UI 최종 검토 체크리스트 |
| `frontend/CLAUDE.md` | Next.js 개발 명령어, 라우트, 패턴 요약 |
| `CODEBASE_REVIEW.md` | 전체 컴포넌트·훅·라우트 구조 |
